# Contributing to QR Code Generator

Thank you for considering contributing to this project!

## Getting Started

1.  **No Installation Required**: This project is built with vanilla HTML, CSS, and JS. You do not need `npm` or `node`.
2.  **Run Locally**: Simply open the `index.html` file in your preferred web browser.

## Development Guidelines

-   **Code Style**: Keep code clean and readable. Use semantic HTML and modern CSS.
-   **CSS**: Please use the CSS variables defined in `:root` for colors and spacing to maintain consistency.
-   **JavaScript**: Use modern ES6+ syntax.

## AI & Automation

If you are using an AI agent to work on this repository:

1.  **Read the Context**: START by reading `AI_CONTEXT.md`. This file contains critical project constraints (like "No Build Tools").
2.  **Update Documentation**: AI agents are expected to keep `CHANGELOG.md` and `README.md` in sync with code changes.

## Pull Requests

1.  Ensure your changes work by manually testing in a browser.
2.  Update the `CHANGELOG.md` with a description of your changes.
3.  Submit your PR!
